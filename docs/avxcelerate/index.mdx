## Overview

Ansys AVxcelerate software provides a simulation solution to support developing, testing, and validating safe autonomous technologies. This simulation solution saves significant time and costs compared to traditional development and testing methods. It allows you to exercise your autonomous vehicle (AV) and advanced driver assistance system (ADAS) software stack in a closed loop, with sensor-accurate synthetic data in either a software-in-the-loop or hardware-in-the-loop context.

Ansys AVxcelerate Autonomy™ software is an end-to-end safety-driven toolchain combining statistics and simulation at scale to perform sensitivity and reliability analysis critical for developing ADAS and autonomous driving (AD) functions.

AVxcelerate Autonomy APIs provide advanced capabilities to automate scenario import, analysis execution at scale on cloud, and results collection. Additionally, it provides methods based on open standard support for rapid and flexible software under test integration. 

Ansys AVxcelerate Sensors™ solution provides realistic testing and validation of sensors for autonomous systems, allowing you to assess the ADAS and AV perception stacks faster than with physical prototypes.

AVxcelerate Sensors APIs allow you to automate the full process to perform simulations from the preparation of the simulation environment to the leveraging of simulation results. 

## OS and supported languages

AVxcelerate Sensors APIs and libraries are available for Windows and Linux in all languages compatible with the gRPC framework.

AVxcelerate Simulation Framework library is available for Linux in C++.
